About MPP-2025

The Department of Mathematics, National Institute of Technology Jamshedpur (NIT Jamshedpur) is proud to organize the Workshop on MATLAB and PYTHON Programming (MPP-2025) in online / virtual mode during May 19-23, 2025. The main objective of MPP 2025 is to make the participants understand about MATLAB and PYTHON programming. The program will also include Hands on Sessions. Most of the major concepts of MATLAB and PYTHON programming will be covered during this workshop.

Course Contents (But Not Limited To)

MATLAB:

Basic operations, Handling Variables, Data Format, Expressions and Matrices
Conditional / Logical Statements, Execution Control, Loops, Writing Functions
Finite and Infinite Sums, Derivatives, Antiderivatives, Integrals
Analytical and Numerical Solutions of ODE's, Data Input / Output in Various Format
2D and 3D Plotting, Modifying plots using property editor, Automating Plots
Polynomials, curve fitting and interpolations, Toolbox

PYTHON:

Basic Operations, Handling Variables, Data Types
Expressions, Conditional / Logical Statements, Control Statements
Operators, String Operations, Control statement, Input / Out statements, Loops
Writing Functions, Basic Libraries (Math, Complex, NumPy, Matplotlib and SymPI)
